# Silo Feed Reader - Home Assistant Add-on Repository

This repository contains the Home Assistant add-on for [Silo](https://github.com/kebabmane/silo-rss), a self-hosted RSS/Atom feed reader with AI-powered daily briefs.

## Installation

1. Navigate to your Home Assistant instance
2. Go to **Settings** -> **Add-ons** -> **Add-on Store**
3. Click the menu (three dots) in the top right corner
4. Select **Repositories**
5. Add this repository URL: `https://github.com/kebabmane/silo-hass-addon`
6. Click **Add**
7. Find "Silo Feed Reader" in the add-on store and click **Install**

## Add-ons in this Repository

### [Silo Feed Reader](./silo)

A self-hosted RSS/Atom feed reader with:
- Clean, modern reading interface
- AI-powered daily brief summaries
- Keyboard shortcuts for power users
- OPML import/export
- Dark mode support
- Mobile-friendly PWA
