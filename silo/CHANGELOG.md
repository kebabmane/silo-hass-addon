# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2025-01-12

### Added

- Initial release of Silo Feed Reader for Home Assistant
- Home Assistant Ingress support (embedded UI in sidebar)
- Single-container deployment with web server and background jobs
- Persistent storage for database and uploads
- Configurable options:
  - Timezone
  - Log level
  - Rails threads
  - LiteLLM integration for AI daily briefs
- Multi-architecture support (aarch64, amd64, armv7, armhf)
- Health check watchdog integration
- Home Assistant backup support
